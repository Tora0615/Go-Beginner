變數型態
===

# 概述 
變數型態分為 : 
* 基礎變數型態 (原生資料型別，Primitive DataType)
    * 整數型 (int)
    * 浮點數型 (float)
    * 複數型 (complex)
    * 布林變數型 (bool)
    * 字符型 (string)
    * 字串型 (byte, rune)
* 複合變數型態 (衍生資料型態\別，Derived DataType)
    * 陣列 (array)
    * 切片 (slice)
    * 映射 (map)
    * 函數 (function)
    * 結構體 (struct)
    * 通道 (channel)
    * 介面 (interface)
    * 指標 (pointer)
